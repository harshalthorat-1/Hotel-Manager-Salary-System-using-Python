# Hotel-Manager-Salary-System-using-Python
🚀 Just built a Python-based Salary Management System for waiters/hotel staff!

💼 Features:
🔹 Auto calculates base pay + overtime
🔹 Adjusts for attendance
🔹 Saves data in Excel files for record-keeping
🔁 Allows multiple entries with clean looping

This project is a great step toward automating HR tasks for small hospitality businesses.

Tech Stack: Python, Pandas, Excel I/O, CLI
